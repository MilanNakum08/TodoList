About This Todo Application

This application was developed using Angular.

It is designed to help you manage your tasks effectively.

  • Components: The app is structured using Angular components for modularity and reusability.
  • Services: Angular services are used for managing data and business logic.
  • Routing: Angular Router is used for navigating between different views such as Home and About pages.
  • LocalStorage: Tasks are stored in the browser's localStorage to persist data across sessions.
  • Bootstrap: The app uses Bootstrap for styling and responsive design.
